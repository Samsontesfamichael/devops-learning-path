### 01: Linux Core and CLI

This module covers the absolute foundations of interacting with Linux operating systems via the command line interface.

### Learning Checklist

*   Essential shell navigation commands (`cd`, `ls`, `pwd`, `mkdir`, `rm`)
*   Understanding Linux file permissions (`chmod`, `chown`)
*   Text editing directly inside the terminal (`vim`, `nano`)
*   Writing basic automation tasks using **Bash scripting**
