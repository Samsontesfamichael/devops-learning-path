### 02: Linux Storage and Processes

Focusing on how Linux handles application workflows, data storage, and operating system daemons.

### Learning Checklist

*   Managing system services via `systemctl` and `systemd`
*   Understanding disk partitions, mounting, and monitoring space (`df`, `du`)
*   Inspecting system processes using `top`, `htop`, and `ps`
*   Locating and debugging runtime errors inside `/var/log/`
